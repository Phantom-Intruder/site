Great! You have just made a commit to the repository of this web page! Don't believe me? Go checkout the repo.

How?

This Angular frontend is connected to a Node/ExpressJs backend which has an endpoint that recieves POST requests. By submitting, you just created a post request with your commit message.

The backend recieved the request and ran a few git commands to push your code to the topic/work-on-site branch.

Go to backend repo

So now what?

You are just about to see a full continous integration and deployment pipeline at work.

The branch you just pushed to is set to trigger an Azure DevOps release pipeline on each push. So as you are reading this, a pipeline would have started running.

This pipeline does three things:

  1. Set up the environment and install the necessary packages
  2. Build the application from the raw source code
  3. Deploy the application


Once the pipeline has finished running, you should be able to reload this page and see the text you just entered added below. This isn't something that is fetched dynamically via a database, or placed temporarily using Angular. Instead, it is harcoded into the html of the source.


If your changes have not been reflected on the site after a few minutes, it could be due to CloudFront showing a cached version of the page, so please head over to bucket website endpoint.
